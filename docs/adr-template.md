# ADR Template

Use este template para documentar decisões arquiteturais (ADR — Architectural Decision Record).

Title: <short, descriptive title>

Status: {Proposed | Accepted | Deprecated}

Date: YYYY-MM-DD

## Context

Explique o contexto que levou à necessidade da decisão. Quais problemas estamos resolvendo? Que restrições existem?

## Decision

Descreva a decisão tomada — o que foi escolhido e por quê. Inclua alternativas consideradas e por que foram rejeitadas.

## Consequences

Descreva os impactos esperados: riscos, migrações, mudanças no contrato público, operações de deploy, testes necessários.
